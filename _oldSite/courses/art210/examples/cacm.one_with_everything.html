One with Everything

##### Prompt Explore your graphics toolset by drawing one of each type of primitive it provides. ##### Response Here's a simple p5.js sketch that draws one of each basic shape provided by p5.js: a point, a line, a triangle, a quad, a rectangle, an ellipse, an arc, a curve (quadratic and Bezier), and a custom polygon. ```javascript function setup() { createCanvas(800, 800); noLoop(); } function draw() { background(220); // Set colors fill(255, 0, 0); stroke(0, 0, 255); strokeWeight(2); // Draw a point point(50, 50); // Draw a line line(100, 100, 200, 200); // Draw a triangle triangle(300, 300, 400, 300, 350, 400); // Draw a quad quad(450, 450, 500, 400, 550, 450, 500, 500); // Draw a rectangle rect(600, 600, 50, 50); // Draw an ellipse ellipse(700, 700, 50, 75); // Draw an arc arc(50, 750, 50, 50, 0, PI/2); // Draw a quadratic curve noFill(); curve(150, 750, 200, 700, 250, 750, 300, 700); // Draw a Bezier curve bezier(350, 750, 400, 700, 450, 750, 500, 700); // Draw a custom polygon (pentagon) beginShape(); vertex(550, 750); vertex(600, 700); vertex(650, 750); vertex(625, 800); vertex(575, 800); endShape(CLOSE); } ``` In this script, `setup()` creates a canvas and sets it to not loop, and `draw()` draws one of each basic shape in different parts of the canvas, using different colors and stroke weights. Each shape is drawn with a fill color of red and a stroke color of blue, and the stroke weight is set to 2. You can experiment with the parameters of each shape and function as needed, such as the size and position of the shapes, the colors and stroke weights, and the angles and vertices of the arcs and polygons. For the arcs, curves, and polygons, you can also try different modes and parameters to see how they affect the shape. ##### Example #2 This example is an attempt to accurately repersent the example shown in the book ```javascript function setup() { createCanvas(800, 800); } function draw() { background(253); let offset = width/10; let x = offset * 2; strokeWeight(8); fill(253); ellipseMode(CENTER); strokeJoin(MITER); //triangle push(); translate(offset, offset); triangle(0, x, x/2, 0, x, x); pop(); //rectangle push(); translate(offset*2+x, offset); rect(0, x/6, x, x*4/6); pop(); // ellipse push(); translate(offset*3+x*2, offset); ellipse(x/2, x/2, x, x); pop(); // line and point push(); translate(offset, offset*2 + x); line(0, 0, x, x); strokeWeight(16); point(0+x/4, x-x/4); point(x-x/4, 0+x/4); strokeWeight(8); pop(); // curve push(); translate(offset*2 + x, offset*2 + x); bezier(0, x, 0, 0, x, x, x, 0); line(0, x, 0, 0); line(x, x, x, 0); pop(); // vertices strokeJoin(ROUND); push(); translate(offset*3 + x*2, offset*2 + x); beginShape(); vertex(x*0/4, x/2); vertex(x*1/4, x); vertex(x*1/4, 0); vertex(x*3/4, x); vertex(x*3/4, 0); vertex(x*4/4, x/2); endShape(); pop(); strokeJoin(MITER); // arc push(); translate(offset, offset*3 + x*2); arc(x/2, x/2, x, x, 0.25*PI, 1.75*PI, PIE); pop(); // quad push(); translate(offset*2 + x, offset*3 + x*2); quad(0, 0, 0, x, x, x, x*2/3, 0); pop(); // ellipse n.2 push(); translate(offset*3 + x*2, offset*3 + x*2); translate(x/2, x/2); rotate(radians(41.4096)); ellipse(0,0, x*1.3333, x*0.6666); pop(); } ```
