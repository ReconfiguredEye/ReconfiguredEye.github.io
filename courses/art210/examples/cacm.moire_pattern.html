Geometric Progression

##### Prompt Generate a set of parallel lines or curves, spaced at narrow intervals. Overlap two copies of your line set, differing by a small rotation, to create a moire pattern. Place some dimension of variability, such as the line separation or rotation angle, under interactive control. ##### Response You can create a moiré pattern by drawing two sets of parallel lines, where the second set is rotated by a small angle relative to the first set. To control the rotation angle interactively, you can use a slider. Here is a simple p5.js sketch that implements this: ```javascript let angleSlider; // Slider to control the rotation angle function setup() { createCanvas(800, 800); angleSlider = createSlider(0, PI/2, PI/8, 0.01); // Slider ranges from 0 to PI/2 with a default value of PI/8 and a step size of 0.01 angleSlider.position(10, 10); } function draw() { background(220); let angle = angleSlider.value(); translate(width/2, height/2); // Center the drawing for(let i = -width/2; i <= width/2; i += 10) { // Draw the first set of lines line(i, -height/2, i, height/2); } rotate(angle); // Rotate the coordinate system for the second set of lines for(let i = -width/2; i <= width/2; i += 10) { // Draw the second set of lines line(i, -height/2, i, height/2); } } ``` In this script, `setup()` creates a canvas and a slider for controlling the rotation angle, and `draw()` draws two sets of parallel lines where the second set is rotated by the value of the slider relative to the first set. The rotation is performed around the center of the canvas, so the moiré pattern appears centered. You can adjust the parameters as needed, such as the spacing between the lines (the step size in the for loops), the range and step size of the slider, and the size and position of the canvas and the slider.
