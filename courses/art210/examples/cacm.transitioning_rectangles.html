Transitioning Rectangles

##### Prompt Recall that any visual property can be linked to a loop variable, not just position. Use iteration to generate a series of rectangles. Your code should simultaneously control several of the rectangles' visual properties, including their position, height, and fill color. ##### Example #1 Here's a simple p5.js sketch that generates a series of rectangles, controlling their position, height, and fill color using a loop variable: ```javascript function setup() { createCanvas(800, 800); background(200); noLoop(); } function draw() { let numberOfRectangles = 10; let margin = width / 10; // 10% margin on each side let availableWidth = width - 2 * margin; let rectangleSpacing = availableWidth / numberOfRectangles; for(let i = 0; i < numberOfRectangles; i++) { let rectangleX = margin + i * rectangleSpacing; let rectangleY = height / 2; let rectangleHeight = map(i, 0, numberOfRectangles - 1, 50, height / 2); // Fill color is a gradient from red to blue let fillColor = color(map(i, 0, numberOfRectangles - 1, 255, 0), 0, map(i, 0, numberOfRectangles - 1, 0, 255)); fill(fillColor); noStroke(); rect(rectangleX, rectangleY, rectangleSpacing, rectangleHeight); } } ``` This code creates a canvas and draws a series of rectangles across the width of the canvas. It uses the loop's counting variable (`i`) to calculate the `x` position of each rectangle, as well as the rectangle's height and fill color. The `map()` function is used to create a gradient effect for the height and color of the rectangles: as `i` increases, the rectangle height increases and the color transitions from red to blue. The `rect()` function is then used to draw each rectangle. --- ##### Example #2 We can center the composition and move it from the canvas edge by adjusting the `rectangleY` and `rectangleHeight` calculations to ensure the rectangles are centered in the canvas and do not touch the edges. Here's how: ```javascript function setup() { createCanvas(800, 800); background(200); noLoop(); } function draw() { let numberOfRectangles = 10; let margin = width / 10; // 10% margin on each side let availableWidth = width - 2 * margin; let rectangleSpacing = availableWidth / numberOfRectangles; for(let i = 0; i < numberOfRectangles; i++) { let rectangleX = margin + i * rectangleSpacing; let rectangleHeight = map(i, 0, numberOfRectangles - 1, 50, height / 2); // Adjust rectangleY to center the rectangles vertically let rectangleY = (height - rectangleHeight) / 2; // Fill color is a gradient from red to blue let fillColor = color(map(i, 0, numberOfRectangles - 1, 255, 0), 0, map(i, 0, numberOfRectangles - 1, 0, 255)); fill(fillColor); noStroke(); rect(rectangleX, rectangleY, rectangleSpacing, rectangleHeight); } } ``` In this updated version, `rectangleY` is calculated as half the difference between the canvas height and the rectangle height. This positions the rectangle vertically in the center of the canvas. Note that this calculation is done inside the loop so that each rectangle's `y` position is adjusted according to its specific height. --- ##### Example #3 This is a representation of what is pictured for this prompt in the book ```javascript function setup() { createCanvas(800, 800); noSmooth(); } function draw() { background(253); strokeWeight(2); //sets the number of rectangles being drawn var nRectangles = 14; for (var i = 0; i < nRectangles; i++){ //finds the x and y value of current rectangle var x0 = 50 + i*50; var y0 = height-50; //rectangle height increases by 25 var rectH = (i+1)*50; //creates a color gradient by making color dependent on i fill(i*(255.0/nRectangles)); //draws the rectangle rect(x0,y0,50,-rectH); } } ```
