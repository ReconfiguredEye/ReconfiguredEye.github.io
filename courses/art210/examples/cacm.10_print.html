10-Print

##### Prompt In each cell of a square grid, choose with random probability to place a diagonal line pointing up or down. The fascinating maze-like patterns that arise are discussed at length in the book 10 Print by Nick Montfort et al. ##### Response Here's an example: ```javascript let cellSize = 20; // Size of each cell let grid; // 2D array to store the grid function setup() { createCanvas(800, 800); // Calculate the number of cells in each direction let cols = floor(width / cellSize); let rows = floor(height / cellSize); // Create and initialize the grid grid = new Array(cols); for (let i = 0; i < cols; i++) { grid[i] = new Array(rows); for (let j = 0; j < rows; j++) { grid[i][j] = random(1) < 0.5; // Choose a diagonal direction randomly } } } function draw() { background(220); stroke(0); // Draw the grid for (let i = 0; i < grid.length; i++) { for (let j = 0; j < grid[i].length; j++) { let x = i * cellSize; let y = j * cellSize; // Draw a line in the diagonal direction chosen for this cell if (grid[i][j]) { line(x, y, x + cellSize, y + cellSize); } else { line(x, y + cellSize, x + cellSize, y); } } } } ``` In this script, `setup()` creates a canvas and initializes a 2D array as the grid, where each cell stores a boolean value indicating the direction of the diagonal line in that cell (true for "/" and false for "\"). `draw()` then draws the grid by drawing a line in each cell in the direction stored in the grid. You can adjust the cell size and the size of the canvas as needed, and you can try different ways of initializing the grid, such as using different probabilities for the two directions or using noise instead of randomness to create more coherent patterns.
