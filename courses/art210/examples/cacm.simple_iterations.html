Simple Iterations: Seven Circles

##### Prompt Use iteration to copy the figure on the left, in which seven circles are positioned across the canvas. The position of each circle should be computed using your loop's counting variable. Make sure the first circle is inset by a margin; it should lie on the edge of the canvas. ##### Example ```javascript function setup() { createCanvas(800, 800); background(200); noLoop(); } function draw() { let numberOfCircles = 7; let margin = width / 10; // 10% margin on each side let availableWidth = width - 2 * margin; let circleSpacing = availableWidth / (numberOfCircles - 1); for(let i = 0; i < numberOfCircles; i++) { let circleX = margin + i * circleSpacing; let circleY = height / 2; ellipse(circleX, circleY, 100); } } ``` This code creates a canvas and draws seven circles across the width of the canvas. It calculates the spacing between circles by taking the available width (total width minus the margins) and dividing it by the number of gaps between the circles (which is one less than the number of circles). It then uses the loop's counting variable (`i`) to calculate the `x` position of each circle. The `y` position is set to `height / 2`, so all circles are horizontally aligned in the middle of the canvas. The `ellipse()` function is then used to draw each circle.
