Color Observation

##### Prompt Carefully observe the color of your shirt, your table, a wall of your room, and the palm of your hand. Proceeding exclusively by modifying numbers in code, and without using a camera or scanning device, reproduce these colors to the best of your ability. ##### Response Here's an example code that can help you accomplish this: ```javascript let inputs = []; // Array to store the input fields let squares = []; // Array to store the squares function setup() { createCanvas(800, 400); noLoop(); // Create input fields for 4 squares for (let i = 0; i < 4; i++) { let inputR = createInput(); inputR.position(20, i * 80 + 20); let inputG = createInput(); inputG.position(20, i * 80 + 60); let inputB = createInput(); inputB.position(20, i * 80 + 100); inputs[i] = [inputR, inputG, inputB]; squares[i] = {x: 300, y: i * 70 + 50}; } } function draw() { background(220); for (let i = 0; i < squares.length; i++) { let r = inputs[i][0].value(); let g = inputs[i][1].value(); let b = inputs[i][2].value(); fill(r, g, b); square(squares[i].x, squares[i].y, 50); } } function mousePressed() { redraw(); } ``` In this script, setup() creates four sets of RGB input fields, each controlling the color of one square, and draw() draws the squares using the current color settings of the input fields. To update the colors of the squares as the input fields are being adjusted, redraw() is called inside mousePressed(), so the sketch is redrawn whenever the mouse is clicked, i.e., when a new color is entered. Just as in the previous example, you can adjust the size and position of the squares and input fields as needed, and you might want to add labels to indicate which square each set of input fields controls, and which color (red, green, or blue) each individual input field controls. Again, make sure to instruct the students to click anywhere on the canvas after entering new color values to see the color change reflected on the squares.
