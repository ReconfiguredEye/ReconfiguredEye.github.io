Draw Your Initials

##### Prompt Draw your initials with primitive shapes and lines. ##### Example: ```javascript function setup() { createCanvas(400, 400); background(220); } function draw() { // "W" line(50, 100, 70, 200); // left slant line(70, 200, 90, 100); // right slant line(90, 100, 110, 200); // left slant line(110, 200, 130, 100); // right slant // "T" line(170, 100, 220, 100); // top line(195, 100, 195, 200); // vertical // "F" line(260, 100, 260, 200); // vertical line(260, 100, 310, 100); // top line(260, 150, 310, 150); // middle } ``` In this script, `setup()` creates a 400x400 canvas and sets the background color. `draw()` uses the `line()` function to draw segments of lines that form the letters "W", "T", and "F". Each call to `line(x1, y1, x2, y2)` draws a line segment from the point `(x1, y1)` to the point `(x2, y2)`. The coordinates are in pixels, and the origin `(0, 0)` is at the top-left corner of the canvas.
